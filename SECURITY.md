# Security

## Scope

- Synthetic demonstration data only.
- No live customer records, watchlist data, sanctions decisions, or bank secrets are stored in this repo.
- The WordPress plugin and static bundle exist to show the operator surface shape, not to replace regulated KYC systems.

## Reporting

If you discover a security issue, open a private advisory or contact the repository owner through GitHub security channels before public disclosure.

## Operational note

- This repo does not process live KYC submissions or production beneficial-owner records.
- Production KYC systems handling sensitive customer data require formal review, access control, retention controls, and deployment hardening before operational use.
